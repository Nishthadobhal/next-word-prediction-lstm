# Next Word Prediction using LSTM

This project implements a **Next Word Prediction Model** using **TensorFlow**, **Keras**, and **LSTM (Long Short-Term Memory)** networks.

The model learns sentence patterns from text data and predicts the next most probable word based on the given input sequence.
